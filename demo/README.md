# Demo Materials

Pre-built dummy data for live webinar demos. Uses a fictional company called **Meridian Software** — a realistic at-risk account that shows each workflow at its best.

## Quick demo commands (run these live)

```bash
# Show churn risk on a struggling account
cd churn-risk-summarizer && python3 local.py

# Show expansion signals from a positive call
cd expansion-signal-detector && python3 local.py

# Show earned ask after a milestone
cd earned-ask && python3 local.py
```

All three use the Meridian Software account from demo/. Output goes to your terminal + Slack if configured.
